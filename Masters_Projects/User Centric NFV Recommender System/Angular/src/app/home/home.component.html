Network Function Virtualization (NFV)

It is an emerging paradigm that enables network operators to virtualize network hardware functions. Using NFV, typical physical network resources such as switches, firewalls, routers, among others can be managed or controlled as virtual network resources and therefore complementing existing cloud hardware resources
Find more details at https://en.wikipedia.org/wiki/Network_function_virtualization

UCNFV

UCNFV is the User Centric approach that enables clients to selectively choose appropriate NFV services of interest. and shows Performance for selected VNF with for that provider.

Inputs - Required VNF with a provider from list, QoS attribute weights
Output - Performance Score of the slected VNF for that provider
